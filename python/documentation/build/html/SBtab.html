SBtab¶

Provides:
  1. Automatic tranlations from SBtab file to Python objects

2. Automatic verification of format and files 2. Easily callable and changeable entries in tables

SBtab is a uniforming spreadsheet format and designed for the use in Systems Biology. Furthermore, it is a useful format to import stored information into Python to process and edit it.

Furthermore, new table types can be easily defined and used directly via the definition table.

How to load tablib object: Use “SBtabTools.openSBtab(filepath)” to create SBtab Python object. Attention: Only ‘tsv’, ‘csv’, ‘ods’ and ‘xls’ are supported.

See specification for further informations.

getRows(table_type='table', inserted=False)[source]¶

Extract the rows of the SBtab, add first column if necessary.

Parameters:

table_type : str (default ‘table’)

Table type of the SBtab table. Only necessary, if first column was set automatically.

inserted : Boolean

True, if mandatory first column was set automatically.
Returns:

value_rows : list

List containing the entries of the SBtab table as string.

Notes

If first column was set automatically, “inserted” is set to True. The entries in the first column are then an abbreviation of the table type and the number of the current row. The abbreviation is set using the first and the last character of the table type. This function removes comment lines and stores them in a list. See specification for further informations.

Transpose SBtab table. Switch columns and rows.

Notes

Some tables need to store the data in the columns, not in rows. This function enables the user to use the same dictionaries features of SBtab without formatting the file by hand.

This is necessary to read the definition table for example.
