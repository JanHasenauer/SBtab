    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h2><a id="user-content-xlsbtab---an-sbtab-excel-2013-add-in" class="anchor" href="#xlsbtab---an-sbtab-excel-2013-add-in" aria-hidden="true"><span class="octicon octicon-link"></span></a>xlSBtab - an SBtab Excel 2013 Add-in</h2>

<p>This project hosts a basic Excel Plugin, for opening an <a href="http://sbml.org">SBML</a> file within Excel and have it be converted to the SBtab format. Alternatively a file edited in the <a href="http://sbtab.net/">SBtab</a> format can be exported as SBML file. </p>

<h3><a id="user-content-dependencies" class="anchor" href="#dependencies" aria-hidden="true"><span class="octicon octicon-link"></span></a>Dependencies</h3>

<p>For the program to work, you will need to have the following installed: </p>

<ul>
<li>a version of <a href="http://python.org">python 2.x</a></li>
<li>the <a href="http://sbml.org/Software/libSBML">libSBML python bindings</a> for that python version</li>
<li>Excel 2013 to run the plugin</li>
<li>a checkout of the <a href="https://github.com/derHahn/SBtab">SBtab repository</a></li>
</ul>

<h3><a id="user-content-installation" class="anchor" href="#installation" aria-hidden="true"><span class="octicon octicon-link"></span></a>Installation</h3>

<p>After you installed the plugin, you will find a new tab called 'SBtab' in the Add-ins ribbon menu. On that tab, click settings to verify the python path, and the SBtab location. </p>

<p>The add in has four options: </p>

<p><a href="https://raw.githubusercontent.com/fbergmann/xlSBtab/master/images/addin_options.png" target="_blank"><img src="https://raw.githubusercontent.com/fbergmann/xlSBtab/master/images/addin_options.png" alt="SBTab options" style="max-width:100%;"></a></p>

<ul>
<li><strong>Import SBML</strong>: imports an SBML file and replaces the contents of the current sheet with SBtab tables representing that file.</li>
<li><strong>Export SBML</strong>: translates the tables of the current sheet back into an SBML file. </li>
<li><strong>Validate</strong>: currently ensures only that the tables have the expected headers. It does not yet validate the entries in the table. </li>
<li><strong>Settings</strong>: specify the path to the python interpreter, and the SBtab script directory. </li>
</ul>

<p>Once a model is imported it will be displayed like in the example below (here BioModel 10 was imported): </p>

<p><a href="https://raw.githubusercontent.com/fbergmann/xlSBtab/master/images/screenshot.png" target="_blank"><img src="https://raw.githubusercontent.com/fbergmann/xlSBtab/master/images/screenshot.png" alt="SBtab of BioModel 10" style="max-width:100%;"></a></p>
